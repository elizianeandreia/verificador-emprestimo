# Verificador de Empréstimo

Este projeto é uma aplicação web simples desenvolvida em HTML, CSS, TypeScript e JavaScript. Ele permite ao usuário verificar se está apto a receber um empréstimo com base na sua renda mensal, score de crédito e tipo de empréstimo desejado.

## Funcionalidades

- Simulação de aprovação de empréstimo
- Validação com base em critérios de renda e score
- Tipos de empréstimo disponíveis:
  - Duplex
  - Condomínio
  - Carro
- Exibição de mensagens dinâmicas com feedback visual
- Tooltip com os requisitos mínimos para aprovação em caso de reprovação

## Tecnologias utilizadas

- HTML5
- CSS3
- TypeScript
- JavaScript (compilado a partir do TypeScript)
